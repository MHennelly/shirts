how it works:

  • you make an order
  • we confirm order with you, we only do on-demand private sales
  • we generate a hash of a unique, secret passphrase
  • we print the shirt and write the passphrase on it
  • we mint an nft of the shirt as indestructible proof we sold it
  • you get the t-shirt and passphrase

why it works:

  • you can only create the hash if you know the secret passphrase
  • this means bootlegs are easily spotted
  • if the tee doesn't have a hash we've minted before then it's definitely bootleg
  • if it does have a hash we've minted before, then it's only legit if the seller can re-create the hash
  • only legit sellers can re-create the hash because only they know the tee's passphrase
  • a tee is only bootleg-safe as long as the passphrase is kept secret by all previous owners
